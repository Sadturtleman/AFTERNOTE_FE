# 마음의기록 작성

카테고리: Mind-Record
설명: 기록을 작성합니다. type 값으로 기록 유형 구분합니다.
Method: POST
URL: /mind-records
header 포함 여부: 포함 O
백엔드 관리자: 김소희
상태: 개발 완료

### Request

**Body parameter**

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| type | 기록 유형 | String | DAILY_QUESTION / DIARY / DEEP_THOUGHT | X | DIARY |
| title | 기록 제목 | String |  | O | 오늘의 기록 |
| content | 기록 내용 | String |  | X | 오늘은 날씨가 정말 좋았다. |
| date | 기록 날짜 | String | yyyy-MM-dd | X | 2026-01-11 |
| isDraft | 임시저장 여부 | Boolean |  | X | false |
| questionId | 데일리 질문 ID | Long | DAILY_QUESTION일 때만 | O | 3 |
| category | 깊은 생각 카테고리 | String | DEEP_THOUGHT일 때만 | O | 나의 가치관 |
- type = DIARY → questionId, category는 null
- type = DAILY_QUESTION → questionId만 포함
- type = DEEP_THOUGHT → category만 포함
- 임시저장(isDraft = true)인 경우에도 content는 빈 문자열("")로 전달될 수 있음

### Response

**Body**

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| recordId | 생성된 기록 ID | Long |  | X | 15 |

**Example**

```jsx
{
  "type": "DIARY",
  "title": "오늘의 기록",
  "content": "오늘은 날씨가 정말 좋았다.",
  "date": "2026-01-11",
  "isDraft": false
}
```

```java
{
  "type": "DAILY_QUESTION",
  "title": "오늘 가장 기억에 남는 순간은?",
  "content": "저녁 노을을 본 순간이 인상 깊었다.",
  "date": "2026-01-10",
  "isDraft": false,
  "questionId": 3
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 기록 작성 성공 |
| 400 | 잘못된 요청 또는 인증 오류 |
