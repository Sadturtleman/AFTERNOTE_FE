# 비밀번호 변경

카테고리: Auth
설명: 비밀번호를 변경합니다.
Method: POST
URL: /auth/password/change
header 포함 여부: 포함 O
백엔드 관리자: 황규운
상태: 개발 완료
프론트 관리자: 정일혁

### Request

```java
{
	"currentPassword" : "qwer1234!"
	"newPassword" : "qwer1234!!"

}
```

### Response

```jsx
{
    "status": 200,
    "code": 0,
    "message": "비밀번호 변경에 성공하였습니다.",
    "data": null,
}
```
