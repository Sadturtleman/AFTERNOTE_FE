# 프로필 수정

카테고리: User
Method: PATCH
URL: /users/me
header 포함 여부: 포함 O
백엔드 관리자: 김소희
상태: 개발 완료

### Request

Header

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| Authorization | 액세스 토큰 | String | Bearer | X | Bearer eyJhbGciOi… |

### Response

| **key** | **설명** | **value 타입** | **옵션** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- | --- |
| name | 사용자 이름 | String |  | O | 김소희 |
| phone | 연락처 | String |  | O | 01012345678 |
| profileImageUrl | 프로필 이미지 URL | String |  | O | https://cdn.example.com/profile/1.png |
- 모든 필드는 선택 값입니다.

**Example**

```jsx
{
  "name": "김소희",
  "profileImageUrl": "https://cdn.example.com/profile/1.png"
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 프로필 수정 성공 |
| 400 | 잘못된 요청 |
| 401 | 인증 필요 |
