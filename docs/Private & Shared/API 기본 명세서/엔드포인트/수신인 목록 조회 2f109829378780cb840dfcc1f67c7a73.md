# 수신인 목록 조회

카테고리: User
설명: 수신인 리스트 목록을 조회합니다.
Method: GET
URL: /users/receivers
header 포함 여부: 포함 O
백엔드 관리자: 김소희
상태: 개발 완료

### Request

**Header**

Authorization: Bearer {accessToken}

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| receivers | 수신인 목록 | Array |  | N |  |
| receivers[].receiverId | 수신인 ID | Long |  | N | 1 |
| receivers[].name | 수신인 이름 | String |  | N | 김소희 |
| receivers[].relation | 수신인과의 관계 | String | ENUM | N
 |  |

**Example**

```jsx
{
  "receivers": [
    {
      "receiverId": 1,
      "name": "김소희",
      "relation": "딸"
    },
    {
      "receiverId": 2,
      "name": "김소희",
      "relation": "조카"
    },
    {
      "receiverId": 3,
      "name": "김소희",
      "relation": "언니"
    }
  ]
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 수신인 목록 조회 성공 |
| 401 | 인증 실패 (토큰 없음 또는 만료) |
| 500 | 서버 내부 오류 |
