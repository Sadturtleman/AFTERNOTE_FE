# 수신인별 애프터노트 목록 조회

카테고리: User
설명: 특정 수신인에게 전달될 애프터노트 출처별 목록을 조회합니다.
Method: GET
URL: /users/receivers/{receiverId}/after-notes
header 포함 여부: 포함 O
백엔드 관리자: 김소희
상태: 시작 전

### Request

**Header**

Authorization: Bearer {accessToken}

**Path Variable**

| **key** | **설명** | **value 타입** | **옵션** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- | --- |
| receiverId | 수신인 ID | Long |  | N | 1 |

### Response

| **key** | **설명** | **value 타입** | **옵션** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- | --- |
| items | 애프터노트 출처 목록 | Array |  | N |  |
| items[].sourceType | 출처 타입 | String ENUM | INSTAGRAM / GALLERY / GUIDE / NAVER_MAIL | N | INSTAGRAM |
| items[].lastUpdatedAt | 해당 출처의 최근 작성일 | LocalDate |  |  |  |

**Example**

```jsx
{
  "items": [
    {
      "sourceType": "INSTAGRAM",
      "lastUpdatedAt": "2025-11-26"
    },
    {
      "sourceType": "GALLERY",
      "lastUpdatedAt": "2025-11-26"
    },
    {
      "sourceType": "GUIDE",
      "lastUpdatedAt": "2025-11-26"
    },
    {
      "sourceType": "NAVER_MAIL",
      "lastUpdatedAt": "2025-11-26"
    }
  ]
}
```

### Status

| **status** | **response content** |
| --- | --- |
| 200 | 애프터노트 목록 조회 성공 |
| 401 | 인증 실패 |
| 403 | 내 수신인이 아님 |
| 404 | 수신인을 찾을 수 없음 |
| 500 | 서버 내부 오류 |
