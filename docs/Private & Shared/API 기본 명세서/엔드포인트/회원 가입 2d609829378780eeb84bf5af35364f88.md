# 회원 가입

카테고리: Auth
설명: 이메일,비밀 번호, 휴대폰 번호로 회원가입을 합니다.
Method: POST
URL: /auth/sign-up
header 포함 여부: 포함 X
백엔드 관리자: 황규운
상태: 개발 완료
프론트 관리자: 정일혁

비밀번호 확인은 프론트 측에서 처리 가능한 걸로 아는데 그걸로 부탁드립니다.

### Request

```java
{
  "email": "user@example.com",
  "password": "strongPassword123!",
  "nickname": "멋진개발자",
  "profileUrl": "https://s3.ap-northeast-2.amazonaws.com/bucket/profile/image.jpg"
}
```

### Response

```jsx
{
    "status": 200,
    "code": 0,
    "message": "회원가입이 완료되었습니다.",
    "data": {
        "userId": 1,
        "email": "user@example.com",
    }
}
```
