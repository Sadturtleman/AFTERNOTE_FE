# 유저 조회

카테고리: 유저
사용자: 관리자
Method: GET
URL: /users/{user_id}
설명: ID를 사용하여 학생을 조회합니다.
FE: 구현 완료
BE: 구현 완료

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |

| userId | 유저 아이디 | Number |  | X | 1 |
| name | 이름 | String |  | X | “HongGilDong” |
| grade | 학년 | Number |  | X | 3 |
| phoneNumber | 전화번호 | String |  | X | “010-1234-5678” |
| birth | 생년월일 | String |  | O | “2003-03-03” |
