# 출결 검색

카테고리: 검색
사용자: 관리자
Method: GET
URL: /search
param: id , date
설명: ID 와 Date를 사용하여 유저의 출결 정보를 검색합니다. 
FE: 시작 전
BE: 구현 완료

### Query parameter

`/api/search?id=1&date=2024-10-20`

1. id
2. date

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| userId | id | String |  | X | 1 |
| name | 이름 | String |  | X | “HongGilDong” |
| grade | 학년 | number |  | X | 3 |
| attendanceItems |  | Array |  | X | [{}] |
| date | 출결 날짜 | String |  | X | “2024-08-08” |
| status | 출결 정보 | String(ENUM) | "ATTEND" |
"ABSENT" |
"ONLINE” |
”NONE” | X | “ABSENT” |
| reason | 이유 | String |  | O |  |

**Example**

```jsx
{
	id: 1,
	name: "HongGilDong",
	grade: 3,
	attendanceItems: [{
		date: "2025-07-03",
		status: "ATTEND",
```
