# 공통 오류 응답

### 공통, 인증/인가 관련 오류 응답

**인증이 필요한 엔드포인트에 유효한 엑세스 토큰 없이 요청 시**

```json
{
    "status": 401,
    "code": 999,
    "message": "인증이 필요합니다. 로그인해주세요.",
    "data": null
}
```

---

**토큰에 할당된 권한이 부족한 경우 (현재는 어플 로직상 권한 문제 발생 X)**

```json
{
    "status": 403,
    "code": 999,
    "message": "권한이 부족합니다.",
    "data": null
}
```

---

**잘못된 엔드포인트(url)로 요청 시**

```json
{
    "status": 404,
    "code": 999,
    "message": "존재하지 않는 엔드포인트입니다.",
    "data": null
}
```

### 토큰 관련 오류 응답

**Authorization header가 없는 경우**

```json
{
    "status": 400,
    "code": 400,
    "message": "Authorization 헤더 미설정",
    "data": null
}
```

---

**요청 Cookie가 미설정 된 경우**

```json
{
    "status": 400,
    "code": 401,
    "message": "쿠키 값 미설정",
    "data": null
}
```

---

**엑세스 토큰 만료**

```json
{
    "status": 400,
    "code": 402,
    "message": "엑세스 토큰 만료",
    "data": null
}
```

---

**유효하지 않은 엑세스 토큰**

```json
{
    "status": 400,
    "code": 403,
    "message": "유효하지 않은 엑세스 토큰",
    "data": null
}
```

---

**엑세스 토큰 타입 미일치**

```json
{
    "status": 400,
    "code": 404,
    "message": "엑세스 토큰 타입 미일치",
    "data": null
}
```

---

**쿠키에 리프레시 토큰 미존재**

```json
{
    "status": 400,
    "code": 405,
    "message": "리프레시 토큰 미설정",
    "data": null
}
```

---

**리프레시 토큰 만료**

```json
{
    "status": 400,
    "code": 406,
    "message": "리프레시 토큰 만료",
    "data": null
}
```

---

**유효하지 않은 리프레시 토큰**

```json
{
    "status": 400,
    "code": 407,
    "message": "유효하지 않은 리프레시 토큰",
    "data": null
}
```

---

**리프레시 토큰 타입 미일치**

```json
{
    "status": 400,
    "code": 408,
    "message": "리프레시 토큰 타입 미일치",
    "data": null
}
```

---

**사용이 제한된 리프레시 토큰**

```json
{
    "status": 400,
    "code": 409,
    "message": "사용이 제한된 리프레시 토큰",
    "data": null
}
```
