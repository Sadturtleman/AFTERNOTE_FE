# Auth API 테스트 체크리스트 (정일혁)

> **담당 범위**: Auth API 4개 (이메일 인증번호 발송, 이메일 인증번호 확인, 회원가입, 로그인)
> **테스트 환경**: 실제 서버 (`https://afternote.kro.kr/`)

---

## 📋 테스트 전 준비사항

### 1. Logcat 필터 설정
- [ ] Logcat 필터: `tag:OkHttp` 또는 `package:com.kuit.afternote`

### 2. 테스트 범위
- 실제 서버로 API 호출 후 UI 동작 및 Logcat 확인
- 단위 테스트는 별도로 실행 (`./gradlew test`)

---

## ✅ 1. 이메일 인증번호 발송 (`POST /auth/email/send`)

### 테스트 (UI + Logcat 확인)
- [ ] 회원가입 화면에서 이메일 주소 입력 (예: `test@example.com`)
- [ ] "인증번호 받기" 버튼 클릭
- [ ] **Logcat 확인**:
  - [ ] `POST /auth/email/send` 요청 로그
  - [ ] 요청 본문: `{"email":"test@example.com"}`
  - [ ] 응답: `200 OK`
  - [ ] 응답 메시지: `"이메일 인증번호가 발송되었습니다"`
- [ ] **UI 확인**:
  - [ ] 성공 메시지가 화면에 표시되는지 확인
  - [ ] 인증번호 입력 필드에 실제로 입력 가능한지 확인 (키보드로 숫자 입력 테스트)
  - [ ] 에러 발생 시 에러 메시지가 화면에 표시되는지 확인
- [ ] 실제 이메일로 인증번호 수신 확인 (서버가 이메일 발송 기능이 있는 경우)

---

## ✅ 2. 이메일 인증번호 확인 (`POST /auth/email/verify`)

### 테스트
- [ ] 이메일 인증번호 발송 성공 후, 인증번호 입력 (실제 서버에서 받은 번호 또는 테스트용)
- [ ] "다음" 버튼 클릭
- [ ] **Logcat 확인**:
  - [ ] `POST /auth/email/verify` 요청 로그
  - [ ] 응답: `200 OK`
- [ ] **UI 확인**:
  - [ ] 다음 단계(주민등록번호 입력 화면)로 이동했는지 확인
  - [ ] 인증 실패 시 에러 메시지 표시 확인
  - [ ] 인증 성공 시 에러 메시지가 표시되지 않는지 확인

---

## ✅ 3. 회원가입 (`POST /auth/sign-up`)

### 테스트
- [ ] 회원가입 플로우 진행 (이메일 인증 → 주민등록번호 → 비밀번호)
- [ ] 회원가입 완료 버튼 클릭
- [ ] **Logcat 확인**: `POST /auth/sign-up` 요청/응답 확인
- [ ] **UI 확인**: 프로필 설정 화면으로 이동, 실패 시 에러 메시지 표시 확인

---

## ✅ 4. 로그인 (`POST /auth/login`)

### 테스트
- [ ] SplashScreen → "로그인하기" 클릭 후 로그인 화면 진입 확인
- [ ] 이메일/비밀번호 입력 후 "로그인" 버튼 클릭
- [ ] **Logcat 확인**: `POST /auth/login` 요청/응답, AccessToken/RefreshToken 확인
- [ ] **UI 확인**: 메인 화면으로 이동, 실패 시 에러 메시지 표시 확인

---

## 🔍 추가 확인 사항

### 에러 처리 테스트
- [ ] 이메일 미입력 시 "인증번호 받기" 클릭 → 에러 메시지 표시 확인
- [ ] 잘못된 인증번호 입력 시 "다음" 클릭 → 에러 메시지 표시 확인
- [ ] 로그인 실패 시 에러 메시지 표시 확인

### UI 상태 확인
- [ ] 인증번호 발송 성공 후 인증번호 입력 필드에 **실제로 입력 가능한지** 확인 (키보드로 숫자 입력 테스트)
- [ ] 로딩 상태 표시 확인 (로딩 인디케이터가 보이는지, 필요 시)
- [ ] 성공/실패 메시지가 **화면에 실제로 표시**되는지 확인 (Logcat만으로는 부족)

### 네비게이션 확인
- [ ] SplashScreen → 회원가입 화면 이동 확인
- [ ] 회원가입 → 프로필 설정 화면 이동 확인
- [ ] 로그인 → 메인 화면 이동 확인
