# 마음의기록 단건 수정화면 조회

카테고리: Mind-Record
설명: 기록 수정 화면을 조회합니다.
Method: GET
URL: /mind-records/{recordId}
header 포함 여부: 포함 O
백엔드 관리자: 김소희
상태: 개발 완료

기록 수정 화면 진입 시 사용되는 API입니다.
기존에 작성된 기록을 수정하기 위해, 기록의 타입에 따라 필요한 데이터를 조회하여 응답합니다.

### Request

**Path parameter**

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| **`recordId`** | 조회할 기록 ID | Long | X |  |  |

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| **`recordId`** | 기록 ID | Long |  | X | 12 |
| **`type`** | 기록 유형 | String | DAILY_QUESTION / DIARY / DEEP_THOUGHT | X | DIARY |
| **`title`** | 기록 제목 | String |  | X | 오늘의 기록 |
| **`content`** | 기록 내용 | String |  | X | 오늘은 날씨가 정말 좋았다!! |
| **`date`** | 기록 날짜 | String | yyyy-MM-dd | X | 2026-01-11 |
| **`isDraft`** | 임시저장 여부 | Boolean |  | X | false |
|  |  |  |  |  |  |
| **`category`** | 깊은 생각 카테고리 | String | DEEP_THOUGHT일 때만 | O | 나의 가치관 |
- type = DEEP_THOUGHT → category만 포함

**Example**

```jsx
{
  "recordId": 12,
  "type": "DIARY",
  "title": "오늘의 기록",
  "content": "오늘은 날씨가 정말 좋았다!!",
  "date": "2026-01-11",
  "isDraft": false
}
```

```jsx
{
  "recordId": 7,
  "type": "DAILY_QUESTION",
  "title": "오늘 가장 기억에 남는 순간은?",
  "content": "저녁 노을을 본 순간이 인상 깊었다.",
  "date": "2026-01-10",
  "isDraft": false,
}
```

| --- | --- |
| 200 |  |
