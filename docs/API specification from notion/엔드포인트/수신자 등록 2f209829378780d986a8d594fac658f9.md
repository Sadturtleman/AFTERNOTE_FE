# 수신자 등록

카테고리: User
설명: 수신인을 등록합니다/
Method: POST
URL: /users/receivers
header 포함 여부: 포함 O
백엔드 관리자: 김소희
상태: 개발 완료

### Request

**Header**

Authorization: Bearer {accessToken}

| **key** | **설명** | **value 타입** | **옵션** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- | --- |
| name | 수신자 이름 | String |  | N | 김지은 |
| phone | 전화번호 | String |  | Y | 010-1234-1234 |
| relation | 사용자와의 관계 | String ENUM |  | N | DAUGHTER |
| email | 이메일 | String |  | Y | afternote123@naver.com |

```jsx
{
  "name": "김지은",
  "phone": "010-1234-1234",
  "relation": "DAUGHTER",
  "email": "jieun@naver.com"
}
```

### Response

| **key** | **설명** | **value 타입** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- |
| receiverId | 생성된 수신자 ID | Long | N | 1 |

**Example**

```jsx
{
  "receiverId": 1
}
```

### Status

| **status** | **response content** |
| --- | --- |
| 201 | 수신자 등록 성공 |
| 400 | 요청 값 검증 실패 |
| 401 | 인증 실패 |
| 500 | 서버 내부 오류 |
