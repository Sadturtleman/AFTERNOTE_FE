# afternote 생성

카테고리: Afternote
Method: POST
URL: /afternotes
header 포함 여부: 포함 O
백엔드 관리자: 황규운
상태: 개발 진행 중

### Request

### SOCIAL 카테고리

**필수 필드**: category, title, processMethod, actions, credentials

**null이어야 하는 필드**: receivers, playlist

```java
{
  "category": "SOCIAL",
  "title": "인스타그램",
  "processMethod": "MEMORIAL",
  "actions": ["계정 비공개 전환", "마지막 게시물 작성"],
  "leaveMessage": "감사했습니다",
  "credentials": {
    "id": "my_insta_id",
    "password": "password123"
  }
  //receivers:null, -> 이 형태도 상관없고 아예 없어도 됌
}
```

### GALLERY 카테고리

**필수 필드**: category, title, processMethod, actions, receivers

**null이어야 하는 필드**: credentials, playlist

```java
{
  "category": "GALLERY",
  "title": "가족 사진",
  "processMethod": "DELETE",
  "actions": ["사진 백업", "클라우드 정리"],
  "leaveMessage": "소중한 추억들",
  "receivers": [
    { "receiverId": 1 },
    { "receiverId": 2 }
  ]
}
```

### PLAYLIST 카테고리

**필수 필드**: category, title, playlist

**null이어야 하는 필드**: processMethod, actions, leaveMessage, credentials, receivers

```java
{
  "category": "PLAYLIST",
  "title": "나의 마지막 플레이리스트",
  "playlist": {
    "atmosphere": "차분하고 조용하게",
    "songs": [
      {
        "title": "보고싶다",
        "artist": "김범수",
        "coverUrl": "https://..."
      }
    ],
    "memorialVideo": {
      "videoUrl": "https://...",
      "thumbnailUrl": "https://..."
    }
  }
}
```

### Response

```jsx
{
    "status": 200,
    "code": 0,
    "message": "애프터노트 생성에 성공하였습니다.",
    "data": {
       "afternote_id" : 3,
    }
}
```
