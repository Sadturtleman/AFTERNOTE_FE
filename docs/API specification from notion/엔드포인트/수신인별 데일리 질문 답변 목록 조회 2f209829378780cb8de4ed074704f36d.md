# 수신인별 데일리 질문 답변 목록 조회

카테고리: User
설명: 특정 수신인에게 전달될 데일리 질문 답변 목록을 최신순으로 조회합니다.
Method: GET
URL: /users/receivers/{receiverId}/daily-questions
header 포함 여부: 포함 O
백엔드 관리자: 김소희
상태: 시작 전

### Request

Authorization: Bearer {accessToken}

**Query parameter**

| **key** | **설명** | **value 타입** | **옵션** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- | --- |
| receiverId | 수신인 ID | Long | N | 1 |  |

### Response

| **key** | **설명** | **value 타입** | **옵션** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- | --- |
| items | 데일리 질문 답변 목록 | Array |  | N |  |
| items[].dailyQuestionAnswerId | 답변 ID | Long |  | N | 130 |
| items[].question | 질문 내용 | String |  | N | 오늘 하루, 누구에게 가장 고마웠나요? |
| items[].answer | 답변 내용 | String |  | N | 아무 말 없이… |
| items[].createdAt | 작성 날짜 | LocalDate |  | N | 2025-10-09 |

**Example**

```jsx
{
  "items": [
    {
      "dailyQuestionAnswerId": 130,
      "question": "오늘 하루, 누구에게 가장 고마웠나요?",
      "answer": "아무 말 없이 그저 나의 곁을 지켜주는 아내가 너무 고맙다.",
      "createdAt": "2025-10-09"
    },
    {
      "dailyQuestionAnswerId": 129,
      "question": "오늘 하루 나를 미소 짓게 한 순간은?",
      "answer": "퇴근길에 본 석양이 너무 예뻤다.",
      "createdAt": "2025-10-08"
    },
    {
      "dailyQuestionAnswerId": 128,
      "question": "오늘 가장 힘들었던 점은?",
      "answer": "업무가 계획대로 되지 않아서 조금 지쳤다.",
      "createdAt": "2025-10-07"
    }
  ]
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 데일리 질문 답변 목록 조회 성공 |
| 401 | 인증 실패 |
| 403 | 내 수신인이 아님 |
| 404 | 수신인을 찾을 수 없음 |
| 500 | 서버 내부 오류 |
