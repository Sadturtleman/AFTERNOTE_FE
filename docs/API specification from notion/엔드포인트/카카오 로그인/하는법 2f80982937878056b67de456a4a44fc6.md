# 하는법..?

카카오 OAuth 2.0을 안드로이드와 서버에 연결하는 구조는 크게 두 가지 방식으로 나뉩니다. 보안과 데이터 관리 측면에서 **서버 연동 방식**이 주로 권장됩니다.

**1. 주요 연동 아키텍처 (권장 방식)**

안드로이드 앱에서 직접 카카오와 통신하기보다, 앱이 받은 **Access Token**을 서버로 전달하여 처리하는 구조가 일반적입니다.

- **Step 1 (안드로이드):** [**Kakao SDK for Android**](https://developers.kakao.com/docs/latest/en/android/getting-started)를 사용하여 로그인을 수행하고, 카카오로부터 `Access Token`을 발급받습니다.
- **Step 2 (안드로이드 → 서버):** 발급받은 `Access Token`을 자체 API 서버로 전송합니다.
- **Step 3 (서버):** 서버는 전달받은 토큰으로 카카오 인증 서버에 사용자 정보를 요청하여 검증합니다.
- **Step 4 (서버 → 안드로이드):** 검증이 완료되면 서버에서 자체 JWT 등을 생성하여 앱에 반환하며 로그인을 마무리합니다.

**2. 준비 사항**

- [**Kakao Developers](https://developers.kakao.com/) 설정:**
    - 애플리케이션 등록 후 **네이티브 앱 키**(안드로이드용)와 **REST API 키**(서버용)를 확보하세요.
    - 안드로이드 플랫폼 등록 시 **키 해시(Key Hash)** 값을 반드시 등록해야 SDK가 작동합니다.
- **안드로이드 설정:** `build.gradle`에 SDK 의존성을 추가하고 `KakaoSdk.init`을 통해 초기화합니다.
- **서버 설정:** Spring Boot 등을 사용한다면 `OAuth2 Client` 라이브러리를 활용해 카카오 API 엔드포인트를 구성할 수 있습니다.

**3. 핵심 API 활용 (Android SDK)**

- **`UserApiClient.instance.loginWithKakaoTalk`**: 카카오톡 앱을 통한 간편 로그인.
- **`UserApiClient.instance.loginWithKakaoAccount`**: 카카오 계정 직접 입력 로그인 (카톡 미설치 시).
