# 수신인별 타임레터 목록 조회

카테고리: User
설명: 특정 수신인에게 전달될 타임레터 목록을 조회합니다.
Method: GET
URL: /users/receivers/{receiverId}/time-letters
header 포함 여부: 포함 O
백엔드 관리자: 김소희
상태: 시작 전

### Request

**Header**

Authorization: Bearer {accessToken}

**Path Variable**

| **key** | **설명** | **value 타입** | **옵션** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- | --- |
| receiverId | 수신인 ID | Long |  | N | 1 |

### Response

| **key** | **설명** | **value 타입** | **옵션** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- | --- |
| items | 타임레터 목록 | Array |  | N |  |
| items[].timeLetterId | 타임레터 ID | Long |  | N | 12 |
| items[].receiverName | 타임레터 수신인 이름 | String |  | N | 박채연 |
| items[].sendAt | 발송 예정일 | LocalDate |  | N | 2027-11-24 |
| items[].title | 타임레터 제목 | String |  | N | 채연아 20번째 생일을 축하해 |
| items[].content | 타임레터 내용 전체 | String |  | N | 네가 태어난 게 엊그제 같은데… |

**Example**

```jsx
{
  "items": [
    {
      "timeLetterId": 12,
      "receiverName": "박채연",
      "sendAt": "2027-11-24",
      "title": "채연아 20번째 생일을 축하해",
      "content": "네가 태어난 게 엊그제 같은데 벌써 스무살이라니...",
    },
    {
      "timeLetterId": 11,
      "receiverName": "박채연",
      "sendAt": "2027-11-24",
      "title": "항상 고맙고 사랑한다",
      "content": "늘 밝게 웃어줘서 고마워. 너의 앞날이 행복하길...",
    },
    {
      "timeLetterId": 10,
      "receiverName": "박채연",
      "sendAt": "2027-11-24",
      "title": "너의 스무 번째 봄을 축복해",
      "content": "너의 모든 봄이 따뜻하길 바란다...",
    }
  ]
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 타임레터 목록 조회 성공 |
| 400 | 잘못된 요청 |
| 401 | 인증 실패 |
| 403 | 접근 권한 없음 |
| 404 | 수신인을 찾을 수 없음 |
| 500 | 서버 내부 오류 |
