# 로그인

카테고리: Auth
설명: email과 비밀 번호로 로그인 합니다.
Method: POST
URL: /auth/login
header 포함 여부: 포함 X
백엔드 관리자: 황규운
상태: 개발 완료
프론트 관리자: 정일혁

![image.png](%EB%A1%9C%EA%B7%B8%EC%9D%B8/image.png)

### Request

```java
{
  "email": "user@example.com",
  "password": "strongPassword123!"
}
```

### Response

```jsx
{
    "status": 200,
    "code": 0,
    "message": "로그인에 성공하였습니다.",
    "data": {
        "refreshToken": "safasfskldfjasdkgwr34tklgfdldfsdckl...",
        "accessToken": "safasfskldfjasdkgwr34tklgfdldfsdckl...",

    }
}
```
