# 수신인 상세 조회

카테고리: User
설명: 특정 수신인의 상세 정보를 조회합니다
Method: GET
URL: /users/receivers/{receiverId}
header 포함 여부: 포함 O
백엔드 관리자: 김소희
상태: 개발 완료

### Request

**Header**

Authorization: Bearer {accessToken}

**Path Variable**

| **key** | **설명** | **타입** |
| --- | --- | --- |
| receiverId | 수신인 ID | Long |

### Response

| 
key
 |

 |

 |

 |

| receiverId | 수신인 ID | Long |  | N | 1 |
| name | 수신인 이름 | String |  | N | 김소희 |

**Example**

```jsx
{
  "receiverId": 1,
  "name": "김소희",
  "relation": "딸",
  "phone": "010-1234-1234",
  "email": "afternote123@naver.com",
  "dailyQuestionCount": 8,
  "timeLetterCount": 12,
  "afterNoteCount": 4
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 수신인 상세 조회 성공 |
| 401 | 인증 실패 |
| 404 | 수신인을 찾을 수 없음 |
| 500 | 서버 내부 오류 |
