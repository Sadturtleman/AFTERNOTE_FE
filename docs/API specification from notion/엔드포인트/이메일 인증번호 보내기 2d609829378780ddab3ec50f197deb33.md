# 이메일 인증번호 보내기

카테고리: Auth
설명: 이메일로 인증 번호를 보냅니다. 
Method: POST
URL: /auth/email/send
header 포함 여부: 포함 X
백엔드 관리자: 황규운
상태: 개발 완료
프론트 관리자: 정일혁

### Request

```java
{
  "email": "gka365@naver.com"
}
```

### Response

```jsx
{
    "status": 200,
    "code": 0,
    "message": "6자리 인증코드가 발송되었습니다.",
    "data": null
}
```

### Status

| status | response content |
| --- | --- |
| 200 | "6자리 인증코드가 발송되었습니다." |
| 400 |  |
