# 이메일 인증번호 확인

카테고리: Auth
설명: 받은 인증 번호로 인증을 합니다.
Method: POST
URL: /auth/email/verify
header 포함 여부: 포함 X
백엔드 관리자: 황규운
상태: 개발 완료
프론트 관리자: 정일혁

### Request

```java
{
  "email": "gka365@naver.com",
  "certificateCode": "123456"
}
```

### Response

```jsx
{
    "status": 200,
    "code": 0,
    "message": "휴대폰 번호 인증에 성공하였습니다.",
    "data": null
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 휴대폰 번호 인증에 성공하였습니다. |
| 400 |  |
