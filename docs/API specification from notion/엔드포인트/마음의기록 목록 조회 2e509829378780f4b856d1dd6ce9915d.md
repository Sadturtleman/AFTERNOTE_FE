# 마음의기록 목록 조회

카테고리: Mind-Record
설명: 기록 목록을 조회합니다. type 파라미터로 데일리 질문, 일기, 깊은생각 구분합니다.
Method: GET
URL: /mind-records
header 포함 여부: 포함 O
백엔드 관리자: 김소희
상태: 개발 완료

![스크린샷 2026-01-11 16.28.39.png](%EB%A7%88%EC%9D%8C%EC%9D%98%EA%B8%B0%EB%A1%9D%20%EB%AA%A9%EB%A1%9D%20%EC%A1%B0%ED%9A%8C/%E1%84%89%E1%85%B3%E1%84%8F%E1%85%B3%E1%84%85%E1%85%B5%E1%86%AB%E1%84%89%E1%85%A3%E1%86%BA_2026-01-11_16.28.39.png)

Mind-Record의 리스트형 및 캘린더형 화면에서 공통으로 사용하는 조회 API입니다.
type 파라미터를 통해 데일리 질문, 일기, 깊은 생각 기록을 구분하여 조회합니다.

### Request

**Query parameter**

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| **`type`** | 기록 유형 | String | DAILY_QUESTION, DIARY, DEEP_THOUGHT
 | X | DIARY |
| **`view`** | 조회 화면 타입 | String | LIST, CALENDAR | O | LIST |
| **`year`** | 연도 (캘린더 조회 시) | Int |  | O | 2025 |
| **`month`** | 월 (캘린더 조회 시) | Int | 1~12 | O | 1 |

<aside>
💡

**GET /mind-records?type=DIARY&view=LIST**

</aside>

- view=CALENDAR일 경우 year, month 사용
    - 해당 월 전체와 달력에 표시된 날짜들을 내려줍니다.
- view=LIST일 경우 date 선택 사항
    - 기록된 전체 항목을 내려줍니다.

### Response

| **key** | **설명** | **value 타입** | **옵션** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- | --- |
| records | 기록 목록 | Array | X |  |  |
| records[].recordId | 기록 ID | Long | X | 12 |  |
| records[].type | 기록 유형 | String | X | DIARY |  |
| records[].title | 제목 | String | O | 오늘의 기록 |  |
| records[].date | 기록 날짜 | String | yyyy-MM-dd | X | 2026-01-11 |
| records[].isDraft | 임시저장 여부 | Boolean | X | false |  |
| markedDates | 기록이 존재하는 날짜 목록 (캘린더용) | Array(String) | O | ["2026-01-03"] |  |
|  |  |  |  |  |  |

**Example**

```jsx
{
  "records": [
    {
      "recordId": 12,
      "type": "DIARY",
      "title": "오늘의 기록",
      "date": "2026-01-11",
      "isDraft": false // 임시저장 여부 (true: 임시저장, false: 정식 저장)
    }
  ],
  "markedDates": ["2026-01-03", "2026-01-11"] // view=CALENDAR일때 캘린더 점 찍기용 날짜 목록
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 기록 목록 조회 성공 |
| 400 | 잘못된 요청 파라미터 또는 인증 오류 |
