# 프로필 조회

카테고리: User
설명: 프로필 수정 진입 시 기존 정보를 표시합니다.
Method: GET
URL: /users/me
header 포함 여부: 포함 O
백엔드 관리자: 김소희
상태: 개발 완료

### Request

Header

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| Authorization | 액세스 토큰 | String | Bearer | X | Bearer eyJhbGciOi… |

### Response

Body

| **ey** | **설명** | **value 타입** | **옵션** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- | --- |
| name | 사용자 이름 | String |  | X | 김소희 |
| email | 이메일 | String |  | X | afternote@example.com |
| phone | 연락처 | String |  | O | 01012345678 |
| profileImageUrl | 프로필 이미지 URL | String |  | O | https://cdn.example.com/profile/1.png |
- phone, profileImageUrl은 미설정 시 null일 수 있습니다.

**Example**

```jsx
{
  "name": "김소희",
  "email": "afternote@example.com",
  "phone": "01012345678",
  "profileImageUrl": "https://cdn.example.com/profile/1.png"
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 프로필 조회 성공 |
| 400 | 잘못된 요청 또는 인증 오류 |
| 401 | 인증 필요 |
