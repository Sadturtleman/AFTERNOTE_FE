# afternote 수정

카테고리: Afternote
Method: PATCH
URL: /afternotes/{afternote_id}
header 포함 여부: 포함 O
백엔드 관리자: 황규운
상태: 개발 진행 중

### Request

## **PATCH (수정) 요청 규칙**

### 공통 규칙

- **카테고리 변경 불가**: category 필드를 보내면 에러 발생
- **수정하지 않을 필드는 보내지 않으면 됨**: 기존 값 유지
- **카테고리에 맞지 않는 필드를 보내면 400 에러**

### SOCIAL 카테고리 수정

**수정 가능**: title, processMethod, actions, leaveMessage, credentials

**null이어야 하는 필드**: receivers, playlist

```java
{
  "title": "수정된 제목",
  "credentials": {
    "password": "new_password"
  }
}
```

### GALLERY 카테고리 수정

**수정 가능**: title, processMethod, actions, leaveMessage, receivers

**null이어야 하는 필드**: credentials, playlist

```java
{
  "receivers": [
    { "receiverId": 3 }
  ]
}
```

### PLAYLIST 카테고리 수정

**수정 가능**: title, playlist

**null이어야 하는 필드**: processMethod, actions, leaveMessage, credentials, receivers

```java
{
  "playlist": {
    "atmosphere": "새로운 분위기"
  }
}
```

### Response

```jsx
{
    "status": 200,
    "code": 0,
    "message": "애프터노트 수정에 성공하였습니다.",
    "data": {
       "afternote_id" : 3,
    }
}
```
