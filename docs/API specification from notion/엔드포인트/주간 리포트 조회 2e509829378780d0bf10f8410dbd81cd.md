# 주간 리포트 조회

카테고리: Mind-Record
Method: GET
URL: /mind-records/weekly
header 포함 여부: 포함 O
상태: 시작 전

![스크린샷 2026-01-11 17.28.06.png](%EC%A3%BC%EA%B0%84%20%EB%A6%AC%ED%8F%AC%ED%8A%B8%20%EC%A1%B0%ED%9A%8C/%E1%84%89%E1%85%B3%E1%84%8F%E1%85%B3%E1%84%85%E1%85%B5%E1%86%AB%E1%84%89%E1%85%A3%E1%86%BA_2026-01-11_17.28.06.png)

선택한 주차에 해당하는 사용자의 기록 목록을 조회하는 API입니다.

해당 주차에 작성된 기록들을 반환합니다.

### Request

**Query parameter**

| **key** | **설명** | **value 타입** | **옵션** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- | --- |
| **`year`** | 조회 연도 | Integer |  | X | 2025 |
| **`week`** | 연중 주차 | Integer | ISO week 기준 | X | 46 |

### Response

records (기록 다시 읽기)

| **key** | **설명** | **value 타입** | **옵션** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- | --- |
| records | 기록 다시 읽기 목록 | Array<Object> |  | X |  |
| records.recordId | 기록 ID | Long |  | X | 12 |
| records.title | 기록 제목 | String |  | X | 오늘 하루, 누구에게 가장 고마웠나요? |
| records.content | 기록 내용 | String |  | X | 아무 말 없이 그저 나의 곁을 지켜주는 아내가 너무 고맙다. |
| records.date | 기록 날짜 | String | yyyy-MM-dd | X | 2025-10-09 |
| records.type | 기록 유형 | String | DAILY_QUESTION / AFTERNOTE / DEEP_THOUGHT |  | DAILY_QUESTION |

**Example**

```jsx
{
  "records": [
    {
      "recordId": 12,
      "type": "DAILY_QUESTION",
      "title": "오늘 하루, 누구에게 가장 고마웠나요?",
      "contentPreview": "아무 말 없이 그저 나의 곁을 지켜주는 아내가 너무 고맙다.",
      "date": "2025-10-09"
    },
    {
      "recordId": 13,
      "type": "AFTERNOTE",
      "title": "오늘의 기록",
      "contentPreview": "날씨가 좋아서 기분이 한결 나아졌다.",
      "date": "2025-10-11"
    }
  ]
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 주간 리포트 조회 성공 |
| 400 | 잘못된 요청 또는 인증 오류 |
