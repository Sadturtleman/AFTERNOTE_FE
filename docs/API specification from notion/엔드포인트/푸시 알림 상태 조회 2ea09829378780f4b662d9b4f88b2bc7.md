# 푸시 알림 상태 조회

카테고리: User
Method: GET
URL: /users/push-settings
header 포함 여부: 포함 O
백엔드 관리자: 김소희
상태: 개발 완료

로그인한 사용자의 푸시 알림 수신 설정의 현재 상태를 조회합니다.

각 알림 종류별로 수신 여부(true/false)를 반환합니다.

### Request

**Header**
Authorization: Bearer {accessToken}

### Response

| **key** | **설명** | **value 타입** | **옵션** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- | --- |
| timeLetter | 타임레터 푸시 알림 수신 여부 | boolean |  | X | true |
| mindRecord | 마음의 기록 푸시 알림 수신 여부 | boolean |  | X | false |
| afterNote | 애프터노트 푸시 알림 수신 여부 | boolean |  | X | true |

**Example**

```jsx
{
  "timeLetter": true,
  "mindRecord": false,
  "afterNote": true
}
```

### Status

| status | response content |
| --- | --- |
| 200 |  |
| 400 |  |
| 401 | 인증되지 않은 사용자 |

**200 OK**

```java
{
  "timeLetter": true,
  "mindRecord": false,
  "afterNote": true
}
```

**401 Unauthorized**

```java
{
  "code": 999,
  "message": "인증이 필요합니다. 로그인해주세요."
}
```
