# 푸시 알림 설정 변경

카테고리: User
Method: PATCH
URL: /users/push-settings
header 포함 여부: 포함 O
백엔드 관리자: 김소희
상태: 개발 완료

로그인한 사용자의 **푸시 알림 수신 설정을 변경합니다.**

클라이언트는 변경된 토글 상태를 포함하여 **각 알림 종류의 최종 수신 여부(true/false)를 서버로 전달**하며, 서버는 전달받은 값을 그대로 저장합니다.

### Request

Header

| **key** | **value** |
| --- | --- |
| Authorization | Bearer {accessToken} |

Body

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| timeLetter | 타임레터 푸시 알림 수신 여부 | boolean |  | X | false |
| mindRecord | 마음의 기록 푸시 알림 수신 여부 | boolean |  | X | true |
| afterNote | 애프터노트 푸시 알림 수신 여부 | boolean |  | X | false |

**Example**

```jsx
{
  "timeLetter": false,
  "mindRecord": true,
  "afterNote": false

```

### Status

| status | response content |
| --- | --- |
| 200 | 설정 변경 성공 |
| 400 | 잘못된 요청 |
| 401 | 인증되지 않은 사용자 |
