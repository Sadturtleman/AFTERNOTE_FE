# 데일리 질문 조회

Method: GET
URL: /daily-question
header 포함 여부: 포함 O
백엔드 관리자: 김소희
상태: 개발 진행 중

질문은 서버에서 날짜 기준으로 하루 1개를 전달해줍니다.

해당 날짜에 이미 노출된 질문이 있으면 기존 질문을 그대로 반환하고, 해당 날짜에 질문이 없으면 서버에서 자동으로 새 질문을 생성 후 변환합니다.

- 날짜 기준은 서버 시간 기준으로 자정(00:00)
