# Cursor AI Resources

이 폴더는 Cursor AI가 프로젝트를 이해하고 작업하는 데 필요한 리소스를 모아둔 곳입니다.

## 폴더 구조

- `design/`: 디자인 파일들 (Figma 등)
  - AI가 UI/UX 디자인을 이해하기 위한 참고 자료
  
- `rules/`: Cursor AI 작업 규칙 및 가이드라인
  - `branch-workflow.md`: 브랜치 작업 규칙
  - `general-guidelines.md`: 일반적인 작업 가이드라인
  - 기타 주제별 가이드라인 파일들

## 사용 목적

이 폴더의 내용은 Cursor AI가 프로젝트 컨텍스트를 더 잘 이해하고, 일관된 작업 방식을 유지하기 위해 사용됩니다.
